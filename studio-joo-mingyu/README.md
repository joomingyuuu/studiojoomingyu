# Studio Joo Mingyu

GitHub Pages용 정적 웹사이트 초안입니다.

## 사용 방법

1. GitHub에서 `studio-joo-mingyu` 저장소를 만듭니다.
2. 이 폴더 안의 파일들을 모두 업로드합니다.
3. Repository Settings → Pages → Deploy from branch → `main` / root 를 선택합니다.
4. 잠시 후 `https://사용자명.github.io/studio-joo-mingyu/` 주소에서 확인합니다.

## 수정할 곳

- `index.html`: 홈 문구와 Current Question
- `manifesto.html`: 스튜디오 철학
- `projects/`: 프로젝트 소개
- `journal/`: 에세이/블로그 글
- `beauty-archive/`: 아름다움 기록
- `style.css`: 전체 디자인
