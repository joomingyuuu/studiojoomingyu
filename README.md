# Studio Joo Mingyu

A quiet editorial-style static website for GitHub Pages.

## Pages
- Home: `index.html`
- Manifesto: `manifesto.html`
- Projects: `projects/index.html`
- Journal: `journal/index.html`
- Beauty Archive: `beauty-archive/index.html`
- About: `about.html`

## How to publish with GitHub Pages
1. Create a GitHub repository.
2. Upload all files in this folder.
3. Go to Settings → Pages.
4. Set source to `main` branch and `/root`.
5. Save and wait for the GitHub Pages URL.

## Editing
- Replace the large image placeholder on the home page with your own image.
- Add images to `assets/images/`.
- Edit text directly in the HTML files.
- Edit the visual style in `style.css`.
